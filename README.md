# 🏙 Smart Grid Energy Intelligence System

An AI-powered web application that analyzes household energy consumption using machine learning techniques and interactive dashboards.

## 🚀 Features

- ⚡ KMeans Clustering (with Auto Cluster Selection)
- 🚨 Anomaly Detection (Isolation Forest)
- 📊 PCA Visualization
- 📡 Real-time Energy Simulation
- 🗺 Interactive Smart City Map
- 📄 PDF Report Generation
- 📈 Cluster Optimization using Silhouette Score

## 🛠 Tech Stack

- Python
- Streamlit
- Scikit-learn
- Pandas, NumPy
- Plotly
- ReportLab

## 📂 How to Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
